== Description ==

= Яндекс.Метрика =

Официальный бесплатный плагин Яндекс.Метрики для WordPress. Поможет установить счетчик Метрики и настроить передачу данных электронной коммерции без работы с кодом сайта. А также передавать данные о просмотре товаров, добавлении в корзину и продажах. 

= Features =

- Официальный плагин Яндекс.Метрики 
- Отслеживание событий электронной коммерции без работы с кодом сайта
- Быстрая установка
- Поддержка версий WordPress, начиная с 5.2.9
- Плановые обновления
- Оперативная служба поддержки

== Список функций == 
- Автоматический поиск уже установленных счетчиков Яндекс.Метрики и автозаполнение настроек по найденным счетчикам
- Быстрое добавление новых счетчиков Яндекс.Метрики. По умолчанию установлены параметры:
	- Электронная коммерция: включена
	- Вебвизор: включен (можно отключить при необходимости)
	- Карта кликов: включена
- Передача событий электронной коммерции в соответствии с [документацией](https://yandex.ru/support/metrica/data/e-commerce.html):
	- Добавление товара в корзину
	- Просмотр карточки товара
	- Удаление товара из корзины
	- Оформление заказа
- Детализация передаваемых данных о товаре в соответствии с [документацией](https://yandex.ru/support/metrica/ecommerce/data.html)
- Выведение лога событий со следующими кодами ошибок:
	- Устаревшая версия WordPress
	- На сайте нет указанной пользователем таксономии бренда 
	- В теме нет хука, необходимого для работы плагина 
	- Номер счетчика содержит нецифровые символы

== Installation ==

= Установка из административной панели WordPress =

1. Зарегистрируйтесь в Яндекс.Метрике и создайте счетчик отслеживания. Если у вас уже есть счетчик, пропустите этот шаг.

2. В административной панели WordPress перейдите в раздел Плагины → Добавить новый.

3. Найдите в поиске плагин Яндекс.Метрики и установите.

4. Перейдите в раздел Плагины → Установленные. Активируйте установленный плагин.

5. Добавьте номер вашего счетчика Метрики в настройках плагина и сохраните изменения.


= Установка с сайта Wordpress.org =

1. Зарегистрируйтесь в Яндекс.Метрике и создайте счетчик отслеживания. Если у вас уже есть счетчик, пропустите этот шаг.

2. Скачайте файл с плагином Яндекс.Метрики с сайта [wordpress.org](https://wordpress.org/plugins/wp-yandex-metrika) или со страницы [Справки Яндекс.Метрики](https://yandex.ru/support/metrica/ecommerce/wordpress.html). 

3. В административной панели WordPress перейдите в раздел Плагины → Добавить новый.

4. Перейдите по кнопке «Загрузить плагин». Выберите скачанный файл с архивом плагина → Установить.

5. Перейдите в раздел административной панели Плагины → Установленные. Активируйте установленный плагин.

6.  Добавьте номер вашего счетчика Метрики в настройках плагина и сохраните изменения.


== Frequently Asked Questions ==

= Как отправить запрос в службу поддержки? =

Посмотрите обсуждаемые темы на вкладке [поддержки](https://wordpress.org/support/plugin/wp-yandex-metrika), возможно, там уже есть решение. Если возникшая проблема с плагином до этого не обсуждалась, обратитесь в [Службу поддержки](https://yandex.ru/support/metrica/troubleshooting.html)

= Почему плагин бесплатный? =

Это официальный плагин Яндекс.Метрики для WordPress. Мы хотим сделать настройку Метрики и электронной коммерции максимально доступной для наших пользователей, поэтому сделали плагин полностью бесплатным. 

= Можно ли использовать два плагина от разных разработчиков? =

Мы рекомендуем использовать официальный плагин Метрики отдельно от решений других разработчиков. Если вы используете официальный плагин вместе со сторонними модулями, это может нарушить работу счетчика Метрики.

= Что такое электронная коммерция? =
 
Электронная коммерция Яндекс.Метрики — это инструмент для аналитики интернет-магазинов. Он дает развернутую статистику о продажах и взаимодействии с товарами на сайте. [Подробнее](https://yandex.ru/promo/metrica/retail)

= Как проверить корректность отправки событий, не дожидаясь отображения данных в панели управления Яндекс.Метрики? =

В адресной строке браузера добавьте к адресу страницы параметр «_ym_debug=1» и загрузите страницу. Пример: http://example.com/?_ym_debug=1
Подробнее в [Справке](https://yandex.ru/support/metrica/ecommerce/check.html)

= Зачем в плагине отдельные поля «Тип бренда» и «Таксономия или произвольное поле бренда»? =

Эти поля нужны для корректной отправки значения бренда (brand) товара.
- Если в поле «Тип бренда» вы выбрали значение «Термин», то в поле «Таксономия или произвольное поле бренда» укажите название таксономии.
- Если в поле «Тип бренда» вы выбрали значение «Произвольное поле», то в поле «Таксономия или произвольное поле бренда» укажите название произвольного поля.

= Что такое Вебвизор в плагине? =

Технология Вебвизор позволяет проанализировать поведение посетителей на сайте на качественно новом уровне. Вы можете воспроизвести их действия в формате видео и узнать, что они делают на каждой странице, как передвигают курсор мыши, кликают по ссылкам. Включите Вебвизор с помощью чек-бокса рядом с полем счетчика.
